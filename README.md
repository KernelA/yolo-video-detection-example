## How to run

Install dependencies:
```
pip install -r. /requirements.txt
```

For development:
```
pip install -r ./requirements.txt -r ./requirements.dev.txt
```

Run:
```
python ./main.py -c <checkpoint_path> -i <video_path> -o <video_path>
```
